### 回答问题（post）
- 项目地址
``````
<项目地址>survey/addAnswer
``````
- 请求参数(json)
``````
[
    {
    "questionId":1, //文字题或打分题，单选多选不需要
    "choiceId":1,   //单选或多选时
    "content":"abc" //文字选项或打分题(一分：1)时，单选多选不需要
    },
            {
    "questionId":1,
    "choiceId":1,
    "content":"abc"
    },
            {
    "questionId":1,
    "choiceId":1,
    "content":"abc"
    },
    {
    "questionId":1,
    "choiceId":1,
    "content":"abc"
    }
]
``````
- 返回参数
``````
成功
"result":"success",
"message":{

         }  
失败
"result":"error",
"message":null                    
``````
