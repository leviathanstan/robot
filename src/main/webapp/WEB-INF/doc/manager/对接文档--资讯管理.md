#### 添加文章（POST）
- 项目地址
``````
<项目地址>information/manager/addInformation
``````
- 请求参数
`````` 
    url,
    img,        //封面
    title,
    guide,
    source,     //来源
    categoryId  //类别
    content[0].page     //分页页数
    content[0].content  //分页正文
    content[1].page
    content[1].content
``````
- 返回参数
``````
成功

#### 修改文章(POST)
- 项目地址
``````
<项目地址>information/manager/updateInformation
``````
- 请求参数
`````` 
//有修改就发，没有就不用，都为可选
id
url,
img,        //封面
title,
guide,
source,     //来源
//内容有修改可以只发有修改的那页数据
content[0].page     //分页页数
content[0].content  //分页正文
content[1].page
content[1].content
``````
- 返回参数
``````
成功
 {
     "result": "success",
     "message": null
 }
 失败
 {
     "result": "error",
     "message": "修改内容不存在"
 }
``````

- 项目地址
``````
   <项目地址>information/manager/deleteInformation
``````
- 请求参数
``````
   ids  //List<Integer>
``````
- 返回参数
``````
 成功
 {
     "result": "success",
     "message": null
 }
 失败

     "result": "error",
     "message": null
}
