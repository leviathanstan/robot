#### 添加产品（post）
```
<项目地址>/product/addConference
```
- 请求参数
```
    {
     id,
     name,
     brand,
     introduction,
     summary,     
     pattern,     //模式
     type,        //种类
     residence,   //地区
     format,
     img,
     load,     //负载
     axis,     //轴
     effectTime,
    }
```
- 返回参数
```
    成功
    {
        "result": "success",
        "message": {
            "id": 20057,
            "price": 0,
            "brand": "234",
            "introduction": null,
            "summary": null,
            "pattern": null,
            "type": null,
            "residence": null,
            "format": null,
            "img": null,
            "company": null,
            "load": null,
            "axis": null,
            "imgs": null,
            "effectTime": null,
            "lastUpdateTime": null
        }
    }
    失败
    {
       "result":"error",
       "message":"" //错误信息
    }      
```
