### 查找、筛选产品（POST）
- 项目地址
```
<项目地址>/product/getProductList
```
- 请求参数
```
    name            //可选，产品名，模糊搜索
    industryId     //可选，行业id
    brandId        //可选，品牌id
    robotId        //可选，机器人种类id
    partsId        //可选，机器人配件id
    pageNum         //可选，默认为1
```
