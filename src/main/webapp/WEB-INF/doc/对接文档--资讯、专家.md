### 获取指定类别的资讯文章的前八条(Get)
- 项目地址
````
<项目地址>/information/getInformationTop/{categoryId}
````
- 请求参数
````
categoryId //1.行业动态 2.咨询热点 3.政策 4.行业报告

````
- 返回参数
````
{
   "result":"success",
   "message":[
               {"id"://文章id
                "title"://题目
                "source"://来源
                "coverImg"://封面图片路径
                "releaseTime"{
                               "date":{ 
                                       "year":
                                       "month":
                                       "day":
                                       }
                               "time":{
                                       "hour":
                                       "minute":
                                       "second":
                                       "nano":
                                       }        
                             }//发表时间
                "type"://类别id             
                }
                {"id":
                ......
                }
                ......
             ]
}                                       
````
