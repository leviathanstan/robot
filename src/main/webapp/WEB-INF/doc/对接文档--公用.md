### 上传图片(post)
- 项目地址
``````
<项目地址>common/uploadImg
``````
- 请求头
``````
 Headers:    Content-Type = multipart/form-data
``````
- 请求参数
``````
 imgs  //文件数组
``````
- 返回参数
``````
成功
"result":"success",
"message":"message": [
                  "E:\\Apache\\Apache24\\htdocs\\face\\img\\2018-11\\0017549a5975426dbe01d1fc147686e9.jpg",

### 上传文件(post)
- 项目地址
``````
<项目地址>common/uploadFile
``````
- 请求头
``````
   Headers:    Content-Type = multipart/form-data
``````
- 请求参数
``````
   files  //文件数组
``````
- 返回参数
``````
成功
"result":"success",
"message":"message": [
                  "E:\\Apache\\Apache24\\htdocs\\face\\img\\2018-11\\0017549a5975426dbe01d1fc147686e9.jpg",
