##获取协会新闻具体信息（GET）
- 项目地址
``````
<项目地址>association/getRobotNewsInf
``````
- 请求参数
``````
 urlId  文章id
``````
- 返回参数
``````
成功
"result":"success",
 "message":{
             "urlId": //文章id
             "url": //文章路径
             "title": //标题
             "amount": //浏览数
             "content": //内容
             "postDate":{
                         "date":{
                                 "year":
                                 "month":
                                 "day":
                                 }
                         "time":{
                                 "hour":
                                 "minute":
                                 "second":
                                 "mamo"
                                 }
                         }//发表时间
             }       
             
 失败
 "result":"error",
 "message":null                    
``````
