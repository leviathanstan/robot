### 获得首页烂协会会长与副会长（GET）
- 项目地址
```
<项目地址>/association/getAssociationMember
```
- 请求参数
```
 无
```
- 返回参数
```
{
    "result": "success",
    "message": [
        {
            "id": 1,
            "memberName": "2",
            "position": "轮值会长",
            "companyId": 1    //所代表公司
        }
    ]
}
```
