#### 登录（post）
```
<项目地址>/user/login
```
- 请求参数
```
    {
       username //String用户名
       password //String密码
    }
```
- 返回参数
```
    成功
    {
       "result":"sucess",
       "message":user //User用户
    }
    失败
    {
       "result":"error",
       "message":null
    }      
```

#### 忘记密码(post),发送验证码是触发
```
- <项目地址> /user/forgetPassword
```
- 请求参数
```
   {
       email //String用户填写的邮箱地址
   }
```
- 返回参数
```
   成功
   {
      "result":"success",
      "message":"已发送验证码到你的邮箱，请验证"
   }
   失败
   {
       "result":"error"
       "message":"该邮箱未注册"
   }
```
#### 忘记密码时邮箱验证(get),确认验证码时触发
```
- <项目地址> /user/validateEmail
```
- 请求参数
```
   {
       checkCode  //String,用户填写的验证码
   }
```
- 返回参数
```
    成功
   {
      "result":"success",
      "message":null
   }
   失败
   {
       "result":"error"
       "message":"输入不能为空"或"验证码错误"
   }
```
