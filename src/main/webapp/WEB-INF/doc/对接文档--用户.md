#### 登录（post）
```
<项目地址>/user/login
```
- 请求参数
```
    {
       username //String用户名
       password //String密码
    }
```
- 返回参数
```
    成功
    {
       "result":"sucess",
       "message":user //User用户
    }
    失败
    {
       "result":"error",
       "message":null
    }      
```

#### 协会审批(POST)
```
- <项目地址> /user/judgeMember
```
- 请求参数
```
- id //会员id
- enterpriseId //企业id
- memberRank //会员等级
- memberIdentifier //会员证号
- associationApproval //协会审批意见
- status //传递是否通过 1 等待审批 2通过审批 3审批失败
```
- 返回参数
```
{
    "success": "success",
    "message": "填写完成"
}
```
