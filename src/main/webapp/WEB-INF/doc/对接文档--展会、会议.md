### 获得相应的展会信息（GET）
- 项目地址
```
<项目地址>/conference/getRelevantConference
```
- 请求参数
```
 relevantId //0 表示将要举办， -1表示已过期
```
- 返回参数
```
{
    "result": "success",
    "message": [
        {
            "id": 1,
            "coverImg": null, //封面图片地址
            "title": "1",
            "holdTime": "",
            "address": "123213",
            "host": "123312", //举办方
            "introduction": "123123", //会议介绍
        }
    ]
}
```
