#### 报名（post）
```
<项目地址>/conference/attend
```
- 请求参数
```
    {
       conferenceId,
       name,
       gender,
       phone,
       weChat,
       email,
       companyName,
       position,
       companyAddress
    }
```
- 返回参数
```
    成功
    {
       "result":"sucess",
       "message":"成功提交"
    }
    失败
    {
       "result":"error",
       "message":"" //错误信息
    }      
```

    }
